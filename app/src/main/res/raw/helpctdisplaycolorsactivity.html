Set Colors

Here you can edit the colors of the items displayed in the Single Chrono/Timer (CT) screen.
These items are the dotmatrix display, the CT control buttons and the back screen.
To select the next item, press the button labelled with the current item name.
To select the next editable color of that item, press the button labelled with the current color name.
To select the next color space, press the button labelled with the current color space (RGB or HSV)
Play with the three sliders (R/G/B (or H/S/V)) to change its color or key in a color value.
You can provide Preset RGB colors for each item.
Exit this screen to return to the Single Chrono/Timer screen.
